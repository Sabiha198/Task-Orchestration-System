#Task Orchestration System

A modern, feature-rich task management application built with vanilla HTML, CSS, and JavaScript. This project demonstrates practical implementation of sorting algorithms, dynamic programming, and responsive web design.

## Features

- **User Authentication** - Secure login and signup with local storage persistence
- **Task Management** - Create, edit, delete, and mark tasks as complete
- **Priority Levels** - Categorize tasks as High, Medium, or Low priority
- **Due Date Tracking** - Set deadlines with date and time
- **Smart Sorting** - Sort tasks using Merge Sort (by name) or Greedy Algorithm (by priority/date)
- **Instant Search** - Filter tasks in real-time using Linear Search
- **LCS Word Game** - Entertainment feature using Longest Common Subsequence algorithm
- **Algorithm Visualizer** - View code, time/space complexity for all algorithms used
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile


Made with dedication for learning and productivity.
